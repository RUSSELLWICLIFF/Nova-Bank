# 🏦 Nova Bank

Nova Bank is a simple desktop banking application built using **Python** and **Tkinter** as part of my internship mini project. It provides a user-friendly graphical interface that allows users to register, log in securely, deposit money, withdraw money, and view their transaction history.

## 📖 Overview

This project demonstrates the fundamentals of desktop application development using Python and Tkinter. It simulates a basic banking system with secure user authentication and essential banking operations through an interactive graphical user interface.

## ✨ Features

- 👤 User Registration
- 🔐 Secure Login
- 💰 Deposit Money
- 💸 Withdraw Money
- 📜 Transaction History
- ✅ Input Validation
- 🖥️ User-Friendly GUI

## 🛠️ Technologies Used

- Python
- Tkinter
- Graphical User Interface (GUI)

## 🚀 Getting Started

### Prerequisites

- Python 3.x

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Nova-Bank.git
```

2. Navigate to the project folder:

```bash
cd Nova-Bank
```

3. Run the application:

```bash
python main.py
```
